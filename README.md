# Soulsborne Build Planner

## Purpose

A web application that allows players to create and plan character builds across the Soulsborne series.

## Target Games

- Elden Ring
- Dark Souls
- Dark Souls 2
- Dark Souls 3
- Bloodborne
- Sekiro

## Planned Features

### Version 1

- Select game
- Enter stats
- Build type detection

### Version 2

- Save builds
- Load builds

### Version 3

- Weapon database
- Scaling recommendations
